## 2. Sign in

After installation, sign in with your Sparkle account.

For CLI:

```bash
sparkle login
sparkle setup
```

For the extension, use the sign-in flow inside your IDE.

## 3. Open your project

Open the repository you want to use with Sparkle.

For CLI usage, run commands from your project root:

```bash
cd path/to/your/project
```

## 4. Start using Sparkle

For the extension, open Sparkle inside your IDE and follow the available actions.

For CLI, verify your setup:

```bash
sparkle status
```

To view available commands:

```bash
sparkle --help
```
